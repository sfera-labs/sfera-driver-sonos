Driver for the integration of the Sonos wireless HiFi system.

@author Giampiero Baggiani @sfera.config type cc.sferalabs.sfera.drivers.sonos.Sonos @sfera.config localhost host name (or IP address) to be used by the players to reach Sfera and send status notifications. If not specified the driver will select a random local address among the available ones. It is therefore recommended to specify one @sfera.config room room name assigned to the player to connect to; if not specified, the driver will connect to the first player found on the network
